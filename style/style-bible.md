# Style Bible: Echoes of the Departed

## 1. Art Goal
Проект должен вызывать у игрока **"меланхоличное любопытство"** — смесь грусти, трепета и острого желания узнать правду. Главное ощущение — **"красивый распад"**, где воспоминания рассыпаются, оставляя после себя эхо.

## 2. Chosen baseline
**Baseline: Теплый сюрреализм (Direction B)**
- Почему: Для нарратива о памяти и потерянных жизнях "холодный" техногенный стиль делает Архив безличной машиной. "Теплый" стиль очеловечивает потерю, добавляет трагизма и красоты.

## 3. Style Rules

| Правило | Описание |
|---------|----------|
| Палитра | Приглушенные тона: сепия, беж, цвет слоновой кости, выцветший бордовый |
| Материалы | Старая бумага, пыль, дымка, ржавый металл, потрескавшаяся краска |
| Свет | Мягкий, рассеянный, с эффектом свечения (bloom), тени не черные |
| Камера | От первого лица, плавная, в моменты распада — эффект старой пленки |
| Стилизация | Полу-реализм с живописным уклоном, формы теряют четкость на краях |
| Читаемость | Ключевые объекты выделены композицией, "шум" не скрывает подсказки |
| Вариативность | Содержание меняется, но обработка (цвета, зернистость) едина |

## 4. Bans / Запреты
-  Яркие неоновые цвета (RGB, Cyan, Magenta)
- Чистые глянцевые поверхности (зеркала, пластик)
-  Гротескное насилие в визуалах
- Симметрия и идеальные геометрические формы
