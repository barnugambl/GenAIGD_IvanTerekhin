# Echoes of the Departed

**Narrative Puzzle-Adventure**, где игрок вселяется в умерших через их последние воспоминания. Каждая смерть открывает новую подсказку — чтобы спасти мир, нужно понять, что случилось на самом деле.

## Кратко о проекте

| Параметр      | Описание                                                                                  |
| ------------- | ----------------------------------------------------------------------------------------- |
| **Жанр**      | Narrative Puzzle-Adventure                                                                |
| **USP**       | Смерть не наказывает, а награждает информацией. Игрок хочет умирать, чтобы узнать больше. |
| **ЦА**        | Любители What Remains of Edith Finch, Return of the Obra Dinn, Outer Wilds                |
| **Платформы** | PC (Windows)                                                                              |
